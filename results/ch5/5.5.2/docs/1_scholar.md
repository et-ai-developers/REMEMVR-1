**1. No Discussion of Practice Effects in Repeated Testing**

- **Why It Matters:** Practice effects in repeated memory testing are well-documented and can MASK true forgetting (improvements from familiarity offset decay) or CREATE SPURIOUS consolidation patterns (if practice effects differ between Early vs Late phases). The piecewise model assumes trajectory changes reflect sleep consolidation, but they might reflect differential practice across phases.
- **Supporting Literature:** Hartanto et al. - **Potential Reviewer Question:** "How do you distinguish genuine sleep-dependent consolidation (memory trace stabilization) from practice-related improvements (task familiarity, strategy refinement, reduced anxiety)? **1. Encoding Quality Differences (Not Consolidation Differences)**

- **Alternative Theory:** Source-destination differences attributed to "differential consolidation" might actually reflect encoding quality differences. If source locations are encoded more richly (free exploration, contextual integration) than destination locations (task-focused placement), observed trajectory differences reflect initial encoding strength rather than consolidation rate.
- **How It Applies:** The piecewise model tests whether Early-phase slope (Days 0-1) differs from Late-phase slope (Days 1-6). If destination memories start weaker at Day 0 due to poorer initial encoding, they MUST decay faster in absolute terms to reach the same Day 6 floor as source memories (proportional decay). This creates a spurious "steeper Early slope" pattern that reflects encoding quality, not consolidation.
- **Key Citation:** Weisberg et al. (2024) showed that spatial memory encoding quality differs substantially based on navigation modality (physical walking > VR walking), even with 1:1 movement mapping. If source encoding (pickup during exploration) is more naturalistic than destination encoding (placement during scripted task), initial quality differences confound consolidation estimates.
- **Why Concept.md Should Address It:** Reviewers will ask whether the piecewise model controls for Day 0 baseline differences. If source and destination have different Day 0 IRT theta distributions (different encoding quality), the Location Type × Phase interaction might reflect "regression to the mean" artifact rather than true consolidation difference.
- **Strength:** MODERATE
- **Suggested Acknowledgment:** "Add to Section 4: Analysis Approach - Discuss whether LMM includes Day 0 IRT theta as covariate (controls for initial encoding quality). **2. Non-Linear Consolidation Models (Alternative to Piecewise)**

- **Alternative Theory:** Memory consolidation may be non-linear and proactive (continuously shaped by anticipated goals) rather than piecewise linear (fixed breakpoint at 48 hours, linear segments). The piecewise assumption imposes a rigid structure that might miss gradual, curved, or multi-phase consolidation patterns.
- **How It Applies:** Jansen & Keil (2020, *Nature Human Behaviour*) propose "non-linear: proactive and non-linear consolidation" where "there is no fixed starting point and no predetermined endpoint in the representational space. The memory is continuously shaped towards anticipated but changing goal states. The trajectory in multidimensional space is non-linear because such continuous changes can lead to increasing or decreasing emphasis on different memory dimensions." If source-destination consolidation follows gradual exponential decay (smooth curve) rather than two-phase linear decay (sharp breakpoint), the piecewise model might impose artificial structure and miss true consolidation differences.
- **Key Citation:** Jansen & Keil (2020) and Mwangi et al.
